# 前端设计插件

生成独特、生产级的前端界面，避免通用的 AI 美学风格。

## 它做什么

Claude 自动在前端工作中使用此技能。创建生产就绪代码：

- 大胆的美学选择
- 独特的排版和色彩调色板
- 高影响力的动画和视觉细节
- 上下文感知的实现

## 用法

```
"为音乐流媒体应用创建仪表板"
"为 AI 安全初创公司构建着陆页"
"设计带暗黑模式的设置面板"
```

Claude 将选择清晰的审美方向并实现对细节的细致关注的生产代码。
