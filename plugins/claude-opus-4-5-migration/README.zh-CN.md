# Claude Opus 4.5 迁移插件

将您的代码和提示从 Sonnet 4.x 和 Opus 4.1 迁移到 Opus 4.5。

## 概述

此技能更新您的代码和提示使其兼容 Opus 4.5。它自动化迁移过程，处理模型字符串、beta 头信息和其他配置细节。如果迁移后在 Opus 4.5 上遇到任何问题，您可以继续使用此技能来调整您的提示。

## 用法

```
"将我的代码库迁移到 Opus 4.5"
```
