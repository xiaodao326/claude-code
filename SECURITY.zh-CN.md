# 安全政策

感谢您帮助我们保持 Claude Code 的安全！

## 报告安全问题

我们系统和用户数据的安全是 Anthropic 的首要任务。我们感谢安全研究人员出于善意识别和报告潜在漏洞的工作。

我们的安全计划由 HackerOne 管理，我们要求通过其[提交表单](https://hackerone.com/anthropic-vdp/reports/new?type=team&report_type=vulnerability)报告此功能中的任何已验证漏洞。

## 漏洞披露计划

我们的漏洞计划指南定义在我们的 [HackerOne 计划页面](https://hackerone.com/anthropic-vdp)上。
